Anatomy of a Curriculum Step

Each step in a curriculum is built from these four block types, mixed in any order:

📖

Scripture Block

A Bible passage displayed inline. Pulls from the Bible API in the creator's chosen translation.

📝

Teaching Block

Free-form text written by the curriculum creator — explanations, context, commentary, insights.

🎥

Video Block

An embedded YouTube video — BibleProject explainers, seminary lectures, sermons.

💬

Discussion Block

Guided questions for personal reflection or group study. Optional facilitator notes included.
